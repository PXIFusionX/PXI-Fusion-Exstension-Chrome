PXI Fusion

Instructions

Console Version

To use the PXI Fusion Cheat Menu via the console:

  • Open the console in Prodigy by pressing Ctrl+Shift+I (or Cmd+Option+I on Mac).
  • Paste the following code into the console:
(function() {
    const scriptUrl = "https://raw.githubusercontent.com/DragonProdHax/PXI/main/PXI%20Fusion";

    fetch(scriptUrl)
        .then(response => response.text())
        .then(code => {
            eval(code);
        })
        .catch(error => console.error("Failed to load the script:", error));
})();
